# 桃園公車站動態追蹤資料

因本人無力仔細維護，歡迎各同好來編輯。

活躍車輛：出車正常、無異狀之車輛。

非活躍車輛：泛指長久不出車、停用、滯廠、待配置等各種須密切關注之車輛。

大台北場站對活躍車輛：station_plates_p.json

大台北場站非活躍車輛：station_plates_inactive_p.json

大台北場站對路線：station_routes_p.json

台中場站對活躍車輛：station_plates_t.json

台中場站對非活躍車輛：station_plates_inactive_t.json

台中場站對路線：station_routes_t.json
